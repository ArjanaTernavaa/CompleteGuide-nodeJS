# eCommerce MVC-nodeJS

This repository provides a comprehensive guide to building a web application using Node.js. The project demonstrates key concepts such as routing, middleware, controllers, and views, following the MVC (Model-View-Controller) architecture.

## Features

- **MVC Architecture**: Structured application using Models, Views, and Controllers.
- **Express.js**: Web framework for handling routes and middleware.
- **EJS**: Templating engine for rendering views.
- **Static Files**: Public directory to serve static assets like CSS.
- **Middleware**: Custom middleware for handling requests and responses.

## Project Structure

- `controllers/`: Contains logic for handling routes.
- `models/`: Defines data models.
- `routes/`: Handles routing for different endpoints.
- `views/`: EJS templates for rendering HTML.
- `public/`: Static files (CSS, images, etc.).
- `middleware/`: Custom middleware functions.
- `util/`: Utility functions used across the application.
- `data/`: Contains sample data files.

## Prerequisites

- Node.js v14 or later
- npm (Node Package Manager)

## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/ArjanaaTernava/MVC-Node.js-ShopApplication.git
    ```
2. Navigate to the project directory:
    ```bash
    cd MVC-Node.js-ShopApplication
    ```
3. Install dependencies:
    ```bash
    npm install
    ```

## Running the Application

1. Start the development server:
    ```bash
    npm start
    ```
2. Open your browser and go to `http://localhost:3000` to view the application.
